Commands

  • start - start a game
  • join - join the game if any
  • fast - fast deploy
  • auto - enable Auto fire
  • random - enable Random fire

Rules

  • One game per room
  • Coordinates is [a-j0-9] i.g. e3
  • Space or comma separated for deployment
  • Carrier size : 5 slots
  • Battleship size : 4 slots
  • Submarine size : 4 slots
  • Cruiser size : 3 slots
  • Patrol Boat size : 2 slots

Battleship.io CLI demo

Battleship.io is a game host project powered by socket.io and node.js. This is a simple cli example demonstrate how the game host works.

For more detail, please visit github.

Hint:

  • Switch room by changing url hash #<room name>. Then you can share this url to your friend to start your own game.
  • If you sick of deploy your fleet one by one, try "fast" deploy, which will deploy your ships to these locations. [a0-a4, b0-b3, c0-c3, d0-d2, e0-e1]
  • If you sick of fire step by step, try "auto" or "random" fire, which will enable automation 'til death.
